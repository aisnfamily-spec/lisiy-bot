Privacy Policy

Лисий Bot — YouTube AI Commentator

Last updated: March 30, 2026

1. Overview

Лисий Bot is a free Chrome extension that helps YouTube creators respond to comments using AI-generated replies. We are committed to protecting your privacy and being transparent about what data the extension handles.

2. Data We Collect

We do not collect, store, or transmit any personal data. The extension collects anonymous usage statistics, such as the number of active installations, to improve the extension. No personal data, account information, or IP addresses are stored.

3. Data Processed Locally

The following data is stored locally in your browser using Chrome Storage API and never leaves your device:

  • Groq API keys — entered by you to enable AI replies. Stored locally, never shared.
  • Extension settings — reply style, language preference, token limit. Stored locally.

4. Data Sent to Third Parties

When you use the AI reply feature, the extension sends the following to the Groq API:

  • The text of the YouTube comment you are replying to
  • The video title and channel name (for context)
  • Your Groq API key (for authentication)

This data is sent directly from your browser to Groq's servers. We do not proxy, log, or store this data. Groq's privacy policy applies to data they receive: groq.com/privacy-policy

5. YouTube Data

The extension reads the following from YouTube pages you visit:

  • Comment text (to generate AI replies)
  • Video title and channel name (for AI context)
  • Comment count (to display in the panel)

This data is processed locally in your browser and is only sent to the Groq API when you explicitly trigger a reply. The extension does not access your YouTube account, credentials, watch history, or any other YouTube data.

6. Permissions Explained

  • youtube.com, studio.youtube.com — to inject the reply panel and AI buttons on comment sections
  • api.groq.com — to send comment text for AI reply generation
  • storage — to save your settings and API keys locally
  • activeTab — to toggle the extension panel from the popup

7. Analytics

The extension sends a minimal anonymous ping to our analytics service to count active installations. No personal data or identifiers are transmitted. There are no tracking pixels, cookies, or ads.

8. Data Security

All data is stored locally in your browser's Chrome Storage. API keys are never transmitted to any server other than the Groq API for authentication purposes.

9. Children's Privacy

The extension is not directed at children under 13. We do not knowingly collect data from children.

10. Changes to This Policy

We may update this policy from time to time. Changes will be reflected on this page with an updated date.
